MotiveBoard is a visual workspace for structuring mystery narratives, designed for writers. 
Comes in handy when you don't have a pen and paper on you, 
you can organize your thoughts on a blank canvas and quickly save, before your flow of ideas ends. 

Features of the app:
- create cards (including character, clue, event, suspect and custom (empty) cards)
- connect elements
- drag and organize
- delete cards
- export as PNG

Installation instructions
1. Clone the repository: git clone https://github.com/ellietsoy/motiveboard

2. Open the project with your IDE (I use Visual Studio Code with Live Server extension)

3. Start the Spring Boot backend (in Visual Studio, I click the "run" button right above the main class of the BackendApplication.java file)
This starts the server on http://localhost:8080

4. (I recommend this):
Open "html_structure.html" with Live Server extension in Visual Studio

5. Use the application
